X3D Standard

X3D (Extensible 3D) is a royalty-free, open standards file format and run-time architecture for representing and communicating 3D scenes and objects. Developed by the Web3D Consortium, X3D serves as the successor to the Virtual Reality Modeling Language (VRML), offering enhanced features and extensibility for real-time interactive 3D graphics on the web. Web3D

For detailed information, you can refer to the official X3D specifications: X3D Specifications
