Composable 3D NFT

This Composable 3D NFT recipe extends the Basic Profile with properties specific to 3D models and objects. This profile combines Schema.org metadata with technical properties from glTF and related standards to define reusable 3D assets.

It uses the hasPart attribute to define the individual parts of the model, so that they can be combined with other NFTs.
